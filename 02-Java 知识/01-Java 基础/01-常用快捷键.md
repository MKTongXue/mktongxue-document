# 1 Windows 系统

## 1.1 快捷键
```text
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + A 全选
Ctrl + X 剪切
Ctrl + Z 撤销
Ctrl + S 保存
Win + R 运行
Win + G 录制屏幕
```


## 1.2 命令提示符窗口
打开方式
```text
开始 + Windows 系统 + 命令提示符
Win 键 + R 输入 cmd（推荐使用）
资源管理器的地址栏前面直接输入 cmd + 路径
```

常用命令
```text
// 盘符切换
C: 或者 D:

// C 盘切换至 E 盘 (当前在 C 盘)
cd /d E:

// 查看当前目录下所有文件
dir

// 切换目录
// change directory
cd
// 返回上个目录
cd..

// 清理屏幕
// clear screen
cls   
// 退出终端
exit

// 查看电脑IP地址
ipconfig

// 打开应用
// 打开计算器
calc
// 打开画图
mspaint
// 打开记事本
notepad

// ping 命令
ping www.baidu.com

// 文件操作
// 创建目录 md mk001
md 目录名
// 删除目录 rd mk001
rd 目录名

// 创建文件 cd>001.txt
cd>文件名
// 删除文件 del 001.txt
del 文件名
```
